# Support

Agentic Security Harness is a pre-release defensive benchmark. Support is best effort.

## Where to ask

- Bugs and reproducible CLI/reporting problems: open a GitHub issue with sanitized
  reproduction steps.
- Defensive pattern proposals: use the "Defensive pattern proposal" issue template.
- Security vulnerabilities in this repository: use GitHub private vulnerability reporting.

## What is supported

- Local deterministic benchmark runs.
- Artifact validation and report interpretation.
- Synthetic pattern proposals.
- Authorized adapter design discussions.

## What is not supported

- Running tests against third-party systems without authorization.
- Handling real secrets, private customer data, or live exploit payloads.
- Treating a clean run as a production security guarantee.
- Public cross-model leaderboards before methodology stabilizes.
